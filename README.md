# Weather App JS

A real-time weather dashboard built with **Vanilla JavaScript**, **HTML5**, and **CSS3**. This application fetches current weather conditions and 24-hour forecasts using the OpenWeatherMap API, featuring a responsive glassmorphism UI.

## 🚀 Features
- **Dynamic Search:** Real-time data fetching for any city worldwide.
- **Detailed Forecast:** Displays 3-hour interval forecasts for the next 24 hours.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Security Focused:** Implements `.gitignore` and template patterns to protect sensitive API credentials.

## 🔒 Security Note
This repository follows industry-standard security practices. The `script.js` file is excluded from version control via `.gitignore` to prevent API key exposure. A `script.js.template` is provided for users to implement their own keys.

## 🛠️ Installation & Setup
1. Clone the repository.
2. Rename `script.js.template` to `script.js`.
3. Obtain a free API Key from [OpenWeatherMap](https://openweathermap.org/).
4. Replace `'YOUR_API_KEY_HERE'` in `script.js` with your actual key.
5. Open `index.html` in your browser.
